1. What is the difference between an adaptive website and a fully responsive website?

Responsive website responds to the size of the screen and displays the page according to the screen size . Responsive Design is used mostly in Mobile and Tablet computers . 

Adaptive Website detects the size of the screen and loads the appropiate layout for it.

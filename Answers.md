1. What is the difference between an adaptive website and a fully responsive website?
ANSWER:Adaptive layout adapts to different screen resolutions and changes the layout based on the resolution. Has       multiple layouts.
Responsive design, responds to screen resolution and shrinks with the screen. Has one layout

2. Describe what it means to be mobile first vs desktop first.
If the website is built with a mobile first in mind, the min-width is used for media queries.
Otherwise max-width is used when prioritizing desktop.

3. What does font-size: 62.5% in the html tag do for us when using rem units?
It makes it easier to use rem measurements, as one rem now equals 10px.

4. How would you describe preprocessing to someone new to CSS?
A neater and better way to write CSS. Without preprocessing the task is doable, but it isn't as reader friendly. it allows for more readability and usually better quality code .

5. What is your favorite concept in preprocessing? What is the concept that gives you the most trouble?
The most favorite would be the ability to replicate html structure. This structure is so much easier to modify with preprocessing rather than just css.
When it comes to nested functions, it gets a bit more complicated but it definitely isn't a hurdle.

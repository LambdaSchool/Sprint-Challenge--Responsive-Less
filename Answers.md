What is the difference between an adaptive website and a fully responsive website?

    An adaptive website is one that is built to specific sized screens, called breakpoints. Each break-point has a different design. The in-between sizes may be hit or miss, because designs may look terrible on size in-between breakpoints.

    A fully responsive website is one that still uses break-points. However, it accounts for the in-between sizes. To allow for good website design across all sizes. The units are primarily in percentages.

Describe what it means to be mobile first vs desktop first.

    A mobile first website is a website the developer builds for mobile devices specifically. The mobile design is the main focus. Designs for bigger screens are secondary designs.

     A desktop first website is a website the developer builds for desktops specifically. The desktop design is the main focus. Designs for smaller screens are secondary designs.

What does font-size: 62.5% in the html tag do for us when using rem units?

    When using rems. It allows for easy conversions between px and rem. With the font-size at 62.5%, 1 rem = 10px.

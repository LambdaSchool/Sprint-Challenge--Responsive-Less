1. What is the difference between an adaptive website and a fully responsive website?

An adaptive website relies on predefined screen sizes to adapt itself to different devices. A responsive website provides a more precise viewing, adapting to ALL screens.

2. Describe what it means to be mobile first vs desktop first.

With mobile first development a website it initially designed from its smallest viewport(min) up towards the largest view. Desktop first starts from the largest view (max) and designs itself down. 

3. What does `font-size: 62.5%` in the `html` tag do for us when using `rem` units?

The 'font-size: 62.5%' allows to easily convert pixel size to rem. It sets the default font px to 10, which is equal to 1 rem.

4. How would you describe preprocessing to someone new to CSS?

Preprocessing allows a developer to take a more intuitive approach to CSS, by using nesting, variables, and mixins ( a preprocessing version of a method) to simplify the the CSS development process.
